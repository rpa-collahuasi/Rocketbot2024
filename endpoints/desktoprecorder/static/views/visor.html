Desktop Recorder

Desktop Recorder captures the actions of a user on the screen and generate the necessary commands to replicate them with Rocketbot.

Recording modes:
  • Rec:   Record multiple actions on the screen
  • Click:   Click on an element
  • Get Text:   Get text in a UI element
  • Select Item:   Select an item in a combobox
  • Set Text:   Modify text in an editable field
  • Send Keys:   Simulates keyboard events
After pressing one of the modes, the recorder will start listening to the clicks. Use the ALT + TAB combination to move between applications.
